# 圖書借閱管理系統

以 `IndexedDB` 為核心的前後台圖書借閱系統，支援學生/教職員共用借還、名冊分流管理、舊系統資料轉換與分年級報表匯出。

## 專案特色

- 前台分頁：`借閱管理`、`查書`
- 後台分頁：`借閱紀錄`、`資料管理（匯入/匯出）`
- 借還流程支援連續掃碼模式（0.5 秒延遲自動送出）
- 借閱紀錄可分頁、排序、刪除（刪歸還或刪整筆）
- 支援舊系統學生/教職員紀錄匯入與唯一識別碼補全
- 支援 `CSV / XLS / XLSX` 書籍清冊匯入
- 提供重置系統（清空全部資料）機制

## 系統檔案

- `index.html`：前台頁面（借閱、查書）
- `admin.html`：後台頁面（紀錄管理、資料匯入匯出）

兩個檔案需放在同一目錄。

## 快速開始

### 方式 1：直接開檔使用

直接開啟 `index.html` 即可使用。

### 方式 2：本機伺服器（建議）

```powershell
python -m http.server 8080
```

開啟：

- `http://localhost:8080/index.html`
- `http://localhost:8080/admin.html`

### 方式 3：GitHub Pages

1. 上傳 `index.html`、`admin.html` 到 repo root
2. 到 `Settings > Pages`
3. Source 設 `Deploy from a branch`
4. Branch 選 `main`，資料夾選 `/(root)`

## 借閱規則

- 學生識別碼：6 碼
- 教職員識別碼：4 碼
- 學生借閱上限：5 本
- 教職員借閱上限：10 本
- 借期：30 天
- 當日借書不可當日還書

## 前台操作說明

### 借閱管理

- 查詢此人：輸入識別碼後查詢借閱中清單
- 查詢此書：輸入登錄號可查看在庫/借出狀態
- 借入、歸還：可啟用連續模式
- 連續模式逾時：10 秒未操作自動取消
- 快速匯出：可直接匯出借閱紀錄（含時間戳）

### 查書

可用以下方式搜尋書籍：

- 登錄號
- 書名關鍵字

顯示 8 欄書目資料，RWD 排版可在窄螢幕完整閱讀。

## 後台功能說明

### 借閱紀錄

- 預設每頁 50 筆
- 依借出時間或還書時間排序（可切換升降冪）
- 可刪除「歸還紀錄」或「整筆借閱紀錄」

### 資料管理（匯入/匯出）

- 學生名冊匯入/匯出
- 教職員名冊匯入/匯出
- 書籍清冊匯入/匯出
- 借閱紀錄匯入（覆蓋全部）/匯出
- 分年級借閱報表匯出
- 舊系統學生借閱紀錄匯入（覆蓋學生）
- 舊系統教職員借閱紀錄匯入（覆蓋教職員）
- 重置系統（清空學生/教職員/書籍/借閱）

## 匯入格式

### 學生名冊 CSV

`年班,座號,姓名,學號`

### 教職員名冊 CSV

`身分證末四碼,姓名,處室,職務`

### 書籍清冊 CSV/XLS/XLSX

欄位：

`登錄號,書名,著者名,ISBN,裝訂別,特藏號,索書號,館藏地,分類號,排架號`

補充：

- 支援全誼原始書庫格式（前兩列為說明、第三列標題、第四列起資料）
- 匯入時會自動偵測標題列並對應欄位

### 借閱紀錄 CSV（覆蓋全部）

`唯一識別碼,處室/班級,職務/座號,姓名,書名,ISBN,登錄號,借出時間,應還時間,還書時間,身分別`

### 舊系統學生借閱紀錄 CSV（覆蓋學生）

`進出日期,學號,登錄號,書名,排架號,狀態`

### 舊系統教職員借閱紀錄 CSV（覆蓋教職員）

`處室/班級,職務/座號,姓名,書名,ISBN,登錄號,借出時間,應還時間,還書時間`

## 資料保存與備份

- 所有資料存於瀏覽器本機 `IndexedDB`
- 不同瀏覽器、不同網域、不同電腦資料彼此不互通
- 建議每日作業結束執行 `匯出借閱紀錄` 備份

## 注意事項

- 匯入功能會檢查標題欄格式，不符會拒絕匯入
- 覆蓋型匯入與重置系統屬不可逆操作，執行前請先匯出備份
- 書籍 `XLS/XLSX` 解析依賴 SheetJS CDN，離線環境建議改用 CSV
